# Accommodation Manager — Web Application

A web application for managing accommodation listings, developed as part of lab exercises for the course *Electronic and Mobile Commerce (2025/2026)*.

## Tech Stack
Java, Spring Boot, React, PostgreSQL

## Description
The system allows users to browse, create, and manage accommodation listings. It demonstrates full-stack development principles with a separated frontend and backend architecture.

## Key Features
- Accommodation listing management (create, view, update, delete)  
- RESTful backend API using Spring Boot  
- React-based frontend for user interaction  
- PostgreSQL database for persistent storage  
- Basic validation and data handling on both client and server side  

## Project Context
This project was developed as part of laboratory exercises for the course *Electronic and Mobile Commerce (2025/2026)*.

## Note
This is an academic project focused on applying full-stack development concepts.
